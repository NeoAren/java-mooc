# Part 01 - 11 - Various Variables

The exercise template contains a program that prints the following:

```
Chicken:
3
Bacon (kg):
5.5
Tractor:
None!

And finally, a summary:
3
5.5
None!
```

Modify the program in the given places so that it outputs the following:

```
Chicken:
9000
Bacon (kg):
0.1
Tractor:
Zetor

And finally, a summary:
9000
0.1
Zetor
```
